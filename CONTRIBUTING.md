# Contributing

## This is NOT an open-source project

The Brussels Governance Monitor is published under a **Source-Available** license.
The code is made public for **transparency and auditability** purposes only.

See [LICENSE](./LICENSE) for full terms.

## Pull Requests

Pull requests are **not accepted** unless a prior written agreement has been made with Advice That SRL.

## Reporting factual errors

If you spot a factual error in the published data, please use the feedback form on the site:

**https://governance.brussels**

## Contact

For any other inquiry: **contact@brusselsgovernance.be**
